# Employee Management System (EMS)

A full-stack **Employee Management System (EMS)** built to manage employees, attendance, leave requests, and role-based access in an organization. This project helped me understand how real-world HR systems work while improving my backend architecture and frontend development skills.

---

## 🚀 Features

- 🔐 Secure Authentication & Authorization
- 👨‍💼 Role-Based Access Control (Admin & Employee)
- 📅 Attendance Management
- 📝 Leave Request System
- 📊 Admin Dashboard
- 👤 Employee Profile Management
- ⚡ Responsive UI with Tailwind CSS
- 🌐 REST API Integration
- ☁️ Deployment Ready Architecture

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
